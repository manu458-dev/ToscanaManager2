| ID | User Story | Priority |
|---|---|---|
| **US-01** | As a **General Director**, I want to **create and configure multiple branches (12) with data separation** so that **I can standardize operations across all locations while maintaining branch-specific control**. | **High** |
| **US-02** | As a **Branch Manager**, I want to **assign role-based permissions with branch-level scope** so that **employees can only access data relevant to their branch and responsibilities**. | **High** |
| **US-03** | As a **Cashier**, I want to **record sales with multiple payment methods (cash, debit, credit)** so that **customers can pay using their preferred method**. | **High** |
| **US-04** | As a **Cashier**, I    want to **apply discounts only with proper authorization** so that **discount fraud is prevented and sales integrity is maintained**. | **High** |
| **US-05** | As a **Cashier**, I want to **automatically generate a ticket/receipt for each transaction** so that **customers receive proof of purchase and all sales are documented**. | **High** |
| **US-06** | As a **User (General Director, Branch Manager, Cashier, Waiter)**, I want to **log into the system using my credentials and be directed to my role-specific interface** so that **I can access only the features and data relevant to my responsibilities**. | **High** |
| **US-07** | As a **Branch Manager**, I want to **execute and validate daily/weekend cash closes** so that **recorded sales match physical cash and discrepancies are identified**. | **High** |
| **US-08** | As a **General Director**, I want to **automatically receive weekly reports per branch and consolidated corporate reports** so that **I can make informed strategic decisions without manual consolidation**. | **High** |
| **US-09** | As a **Cashier**, I want to **perform a cash register closing at the end of my shift** so that **all transactions are reconciled and I can hand over responsibility to the next shift or management**. | **High** |
| **US-10** | As a **Waiter**, I want to **generate table bills with accurate totals** so that **customers can review and pay for their orders efficiently**. | **Medium** |
| **US-11** | As a **Branch Manager**, I want to **create, update, and deactivate user accounts for my branch** so that **only authorized employees have system access**. | **High** |
| **US-12** | As a **General Director**, I want to **manage the product catalog centrally (create, update, logical delete)** so that **product information is consistent and traceable across all branches**. | **High** |
| **US-13** | As a **Diner**, I want to **browse the product catalog with prices and availability** so that **I can make informed purchasing decisions**. | **Medium** |
| **US-14** | As a **Cashier**, I want to **perform cancellations and returns only with authorization and documented reasons** so that **all transaction reversals are legitimate and auditable**. | **High** |
| **US-15** | As an **Administrator**, I want **the system to operate with a local-first architecture** so that **sales continue uninterrupted during internet outages**. | **High** |
| **US-16** | As a **General Director**, I want to **export weekly reports in multiple formats (PDF, Excel, CSV)** so that **I can share insights with stakeholders and perform additional analysis**. | **Medium** |
| **US-17** | As a **Waiter**, I want to **register customer complaints and service notes with details** so that **management is aware of service issues and can address them appropriately**. | **Medium** |
| **US-18** | As a **General Director**, I want to **filter and view reports by my specific branch or across all branches** so that **I can analyze performance at different organizational levels**. | **High** |
| **US-19** | As a **Diner**, I want to **check product availability in the catalog for specific branches** so that **I know which products are available at my preferred location before visiting**. | **Medium** |
| **US-20** | As a **Waiter**, I want to **add products and modifiers to a specific table's order so that the kitchen receives clear instructions and the final bill reflects actual consumption**. | **High** |
